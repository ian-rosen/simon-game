# Simon Game

A browser-based memory game where players repeat an increasingly complex color sequence, testing focus, pattern recognition, and reaction under pressure.

## Overview
Simon Game recreates the classic memory challenge with interactive visuals and game-state logic in JavaScript. The project focuses on event-driven programming, sequence tracking, and user feedback loops. It is a great example of building game mechanics with a clean separation between UI behavior and gameplay rules.

## Features
- Progressive difficulty with longer color sequences each round
- Input validation to detect correct and incorrect user patterns
- Immediate game-over feedback and restart flow

## Technologies Used
- **Languages:** HTML, CSS, JavaScript
- **Frameworks/Libraries:** None
- **Tools:** Node.js, npm, Jest, Git, VS Code

## How to Use

### Local Setup
```bash
git clone [repo-url]
cd simon-game
npm install
npm test
```

Open `index.html` in your browser to play the game.
